Hier ein paar INFOS:
	• Das game wird mit dem Menü gestartet
	    --> Start --> startet von Level 0 und geht weiter
	    --> Levels --> falls man ein bestimmtes Level spielen möchte
	• Steuerung:
	        Pfeiltasten zum Bewegen
			Leertaste zum pausieren
			R zum neustarten
			Esc zum beenden
	• Zurzeit funktioniert das Game so, dass das nächste 
	  Level startet, sobald ihr eine bestimmte Anzahl an
	  Äpfel gegessen habt (10)
